# Vagrant Mono Box

Run [Mono][] in virtual box using [Vagrant][].

## Getting Started

### Dependencies

1. Install [Vagrant][]
1. Install [Puppet][]
1. Install [VirtualBox][]

### How to start

#### Website
1. Publish c# code locally
1. Move publish folder to the `mono` folder at the root level
1. Run `vagrant up`
1. Goto `localhost:8080` to see browse the site
